🔥 Termux Ultimate Setup Tool

«⚡ Powerful all-in-one Termux setup script for installing essential packages, development utilities, and networking tools automatically.»

---

🚀 Features

✔️ Automatic Termux update & upgrade
✔️ Install Python & pip
✔️ Networking & utility tools
✔️ Development environment setup
✔️ Beginner-friendly interface
✔️ Fast one-click installation
✔️ Lightweight & simple to use

📥 Installation

pkg update && pkg upgrade -y

pkg install git -y

git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git

cd setup.sh

python setup.sh
